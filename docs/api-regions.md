# LifeSmart API Regions

Use the table below to choose the `LifeSmart Account Country/Region` value during setup.

LifeSmart's API documentation uses service codes such as `AME`, `EUR`, and `APZ`.
The Home Assistant integration lets you select a country or region directly.
The API region value is resolved automatically.

## Server Endpoints

| Service code | Service area | API region value | API URL |
| --- | --- | --- | --- |
| `GS` | Global/default | Leave empty/default | `https://api.ilifesmart.com/app/` |
| `CN0` | China old | `cn0` | `https://api.cn0.ilifesmart.com/app/` |
| `VIP1` | China special | `cn1` | `https://api.cn1.ilifesmart.com/app/` |
| `CN2` | China new | `cn2` | `https://api.cn2.ilifesmart.com/app/` |
| `AME` | America | `us` | `https://api.us.ilifesmart.com/app/` |
| `EUR` | Europe | `eur` | `https://api.eur.ilifesmart.com/app/` |
| `JAP` | Japan | `jp` | `https://api.jp.ilifesmart.com/app/` |
| `APZ` | Asia Pacific | `apz` | `https://api.apz.ilifesmart.com/app/` |
